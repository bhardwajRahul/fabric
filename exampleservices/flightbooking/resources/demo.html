The Flight Booking example demonstrates an agentic workflow with a real human-in-the-loop pause (Interrupt/Resume), a goto loop for keep-searching, and a child LLM workflow invoked as an isolated subgraph to choose a seat. Seat selection needs a real LLM provider; without one the booking still completes with the first available seat.
