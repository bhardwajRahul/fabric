Embedder Demo

This microservice uses venv.py.core to load a sentence-transformers model in Python and serves Go endpoints that call it. The venv must be initialized before the embedding and similarity actions are available.

Initialize Python VM

Clicking the button below provisions a Python virtual environment in venv.py.core, installs sentence-transformers, and loads the all-MiniLM-L6-v2 model. First run downloads ~80MB and takes 20-60 seconds; subsequent runs reuse the cached venv and complete in a few seconds.

Similarity

Computes cosine similarity between the embeddings of two strings. Range is [-1.0, 1.0]; higher means more semantically similar.

Embedding

Returns the raw embedding vector. The MiniLM model produces 384-dimensional vectors; we show the first eight values.

Python Terminal

Tailed stdout + stderr from the Python venv. Empty until the venv is initialized; updates every second while the page is open.
